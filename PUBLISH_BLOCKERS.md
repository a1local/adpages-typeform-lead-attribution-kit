# Publish Blockers

Current status: publishable as a free local setup kit, checklist, or implementation resource. It is not ready to be marketed as a Typeform app, official integration, OAuth app, webhook receiver, or live lead-sync product.

## Must Be Resolved Before Real Integration Claims

- No Typeform app, partner listing, OAuth flow, app manifest, webhook endpoint, or hosted backend is included.
- No automatic form-response reads, webhook handling, or lead writes exist.
- No Typeform workspace, form, answer ref, hidden field, respondent notification, or connected app has been tested.
- No API scopes, account permissions, webhook signing, token storage, retry handling, rate-limit handling, or support process has been reviewed.
- Hidden fields and answer refs are account-specific and must be verified manually in each target Typeform.
- Consent wording must be reviewed for the target market before production use.

## Acceptable Current Positioning

- "Free Typeform hidden-field attribution setup kit."
- "Typeform UTM and click-ID QA checklist."
- "Local JSON mapper for Typeform lead-attribution previews."
- "Agency resource for Typeform lead tracking handoff."

## Avoid

- "Typeform app."
- "Official Typeform integration."
- "One-click install."
- "Automatically syncs Typeform leads."
- "Webhook processor."
- "Certified" or "partner approved."
